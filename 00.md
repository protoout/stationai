## 1. 今回触れる技術を選ぶにあたり

今回のハンズオンではAIと共にモバイルアプリを作ります。

前半ではManusを使ってモバイルアプリを作り、後半ではManusで作ったものを引き継いでバイブコーディングでの改善体験という流れです。

`AIエージェントツール`と`コーディングエージェントツール`のハイブリッドなハンズオンとなり、現状のAIによるアプリ開発文脈だとホットな内容となっています。

今回触れる`React Native`という技術は昔からあるWebアプリとスマートフォンアプリを作れる技術ですが、`Expo`を使うことで実機での検証ビルド、ストアへの申請などが凄くやりやすくなりました。

（通常iOSだとiPhone + AppStoreなどですが、従来手順だと実機検証や審査・申請プロセスの手間が大きい）

審査プロセスに掛かる時間もあるので、今日のハンズオン時間だけではストアでリリースは厳しいですが、やれる方はぜひ持ち帰ってトライしてみてください。

## 2. 全体の流れと到達目標

- 前半:
  - Manusで`旅行記録アプリ`の作成
  - 作成したアプリをスマホ上で動かす ← **ここまでがいったん全員やりたい**
- 後半:
  - Gemini CLIで開発を引き継ぐ
  - 機能追加を行う
- 応用（時間次第）
  - さらにもう1機能追加してみる
  - `例` アプリがオフの状態でも一定時間ごとに位置情報を取得。地図上に軌跡を表示する
- 制作タイムと発表
  - アイデアワークと応用実装
  - 実装したプロトタイプの発表
- クロージング

## 4 今回利用するツール

オープニングスライドにありましたがおさらい。

### Manus　（前半のみ）

AIエージェントツールです。

チャットで依頼をするとスライドの作成、webサイトの作成、メール送信など色々なツールを組み合わせてタスク実行をしてくれます。

最近のバージョンでアプリ開発 => ストアリリースまでがすごく簡単にいけるようになったと話題になっています。

### React Native + Expo

- React Native

Webアプリケーションを作成する`React`というフレームワークを元にネイティブアプリ（iOSやAndroidなど）を作れるようにしたフレームワーク・ツールです。

Web開発の手軽さでWebアプリ・iOSアプリ・Androidアプリを作れてお得感があります。

- Expo（Expo Go）

`Expo`はReact Nativeを利用してモバイルアプリケーションを開発するためのツールセットおよびプラットフォームです。iPhone / Android の両方に対応していて、アプリ作成後の面倒な部分を色々と吸収してくれます。

`Expo Go`は検証用のスマホアプリの名前です。
作っているアプリをすぐにスマホで確認できます。

参考: https://zenn.dev/keisuke333/books/109bfd5bf5ddf9/viewer/47d139

### GitHub Codespaces　（後半のみ）

- 環境（≒パソコン）とエディターがセットになっているツール
  - VS Codeというエディター
- ブラウザとGitHubアカウントだけあればOK

### Gemini CLI （後半のみ）

Google公式ツールで、Geminiのモデルに依頼をしてソースコードの変更やコマンド実行をしてくれるツールです。

> <img alt="gyazo.png" src="https://i.gyazo.com/a5202ab3c296c325754f960b5632aeab.png" width="450px" />
