# 0. 本日のハンズオンについて（10 min　/ 10 min）

## 0.1. ハンズオン前に: 全体の流れと到達目標

以下の流れでハンズオンをします。

- 本日利用するサービスの確認・準備（詳細は後述）
  - GitHub CodespacesとGemini CLI
  - Manus
  - Expo Go
- **開発スタート** `旅行記録アプリ`の作成
  - `1.` 日時を取得してメモを残せる
  - `2.` 写真を撮影して位置情報を残せる
- 作成したアプリをスマホ上で動かす ← **ここまでがいったんゴール**
- 時間次第: さらにもう1機能追加してみる
  - `例` アプリがオフの状態でも一定時間ごとに位置情報を取得。地図上に軌跡を表示する
- アイデアワークと応用実装
- 実装したプロトタイプの発表
- クロージング

### 0.1.1. 諸注意と進め方と心構え

#### 生成AIの（ざっくりした）仕組みとAIの精度問題

- 文字の次に何が来そうか？のそれっぽいものを生成
  - 例: "大手町"の次に何が来そうか？の一番それっぽいテキストを生成している
    - AI「"大手町..."と来たら駅が続きそうだな」
    > <img src="https://i.gyazo.com/b912198d219ea382d6ee5ab857dd5c10.png" alt="gyazo.png" width="450px" />
- 基本として、AIは再現性が低いです
  - 教科書通りにはいきません

- **AIに期待しすぎないように**
  - デジタルツール全体的にですが銀の弾丸ではないです
- バイブコーディングにも期待しすぎないように
  - コーディング経験がベースにあるとかなり乗りこなせるが、未経験者はツールを使うところで覚えることが多くて大変です
    - 時間を掛ければ乗りこなせます。はじめての人は、今日の時間ですべて覚えられるとは思わないこと
  - 例: AIの自動運転車にあなたが乗っていて、事故起こしそうだったり道を間違えそうになったときにハンドルを握って手動に切り替えられるかどうか
    - 交通ルール知識、車の知識、運転知識や経験などがほしいですよね

#### 再現性を上げるために

指示のある文章はなるべく`コピペ`しましょう

例:

```sh
簡単なオウム返しのLINE Botを作成してください。Node.jsで作成をお願いします。
```

と資料に書いてあるのに`オウム返しLINE Botを作って`と命令をしてしまう

- 指示がない限り、最初は設定も資料通りになぞる
  - 少し変更したい場合は応用時に変更を試しましょう

以下資料において、◻︎2つのアイコン、  

> 赤枠内側のアイコンをクリックすると、グレーの中身全体がコピーできます。  
>
> <img src="https://i.gyazo.com/3219d8e5fc7120368858a22275f7361f.png" alt="gyazo.png" width="450px" />

このアイコンをクリックすることで全体をコピーできます。  
**選択してコピーしようとすると、全文をコピーできずに失敗してしまうことにつながります**ので、ぜひ活用していきましょう

#### 時間は少ないので理解できない、付いていけないもありえます

- ハンズオンは180分程度なのでちょっとしたことしかできないと思います
  - **仕組みを理解ではなく今日は、新しいことを学べたというキッカケを得たという認識で**。しっかり理解したい人や続きをやりたい人は受講後に自分で
  - 上手くいかない人はたぶん出ます... 残念ながらAIの挙動は制御仕切れないのでそれくらいの心持ちでいましょう
- 全体の進捗雰囲気を見ながら前で進行確認をします
  - 6~7割程度目安
  - **TA（ティーチングアシスタント）のスタッフが1名いますので気軽に手を上げて読んでください**
    - 置いていってしまわないといけないときはTAサポートを依頼します

### 0.2.1. GitHub Codespaces + Gemini CLI

- GitHub Codespaces
  - 環境（≒パソコン）とエディターがセット
  - VS Codeを使える
  - ブラウザとGitHubアカウントだけあればOK
- Gemini CLI
  - エディター制御のツール
  - コマンドも実行可能
  - **Google公式ツール**
  - 内部ではgoogleが開発した生成AI **Geminiのモデル（脳）** が使われる

> <img alt="gyazo.png" src="https://i.gyazo.com/a5202ab3c296c325754f960b5632aeab.png" width="450px" />

#### 便利な反面 安全な実行が求められる

- AIが色々環境壊しちゃう問題
- PCにインストールはセキュリティ的にもそもそも厳しい？
- **GitHub Codespacesは仮想のマシンなので壊してしまっても大丈夫**です。学習に良い
