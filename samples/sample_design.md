# 天気アプリ デザイン計画書

## アプリ概要
現在の天気と1週間の天気予報を表示するシンプルな天気アプリ。ダミーデータを使用したプロトタイプ。

## 画面構成

### 1. 現在の天気画面 (Home)
**主要コンテンツ:**
- 現在地の都市名
- 現在の気温(大きく表示)
- 天気アイコン(晴れ、曇り、雨など)
- 天気の説明文(例: "晴れ")
- 体感温度
- 湿度
- 風速
- 最高気温/最低気温

**レイアウト:**
- 上部: 都市名
- 中央: 大きな天気アイコンと気温
- 下部: 詳細情報カード(体感温度、湿度、風速)

### 2. 週間天気予報画面 (Forecast)
**主要コンテンツ:**
- 7日間の天気予報リスト
- 各日の情報:
  - 日付(曜日)
  - 天気アイコン
  - 最高気温/最低気温
  - 降水確率

**レイアウト:**
- スクロール可能なリスト形式
- 各日をカード形式で表示
- 左側: 日付と曜日
- 中央: 天気アイコン
- 右側: 気温と降水確率

## ユーザーフロー
1. アプリ起動 → 現在の天気画面が表示される
2. タブバーで「週間予報」タブをタップ → 週間天気予報画面に遷移
3. タブバーで「現在」タブをタップ → 現在の天気画面に戻る

## カラーパレット
- **Primary**: `#0a7ea4` (青系 - 空をイメージ)
- **Background (Light)**: `#ffffff`
- **Background (Dark)**: `#151718`
- **Surface (Light)**: `#f5f5f5`
- **Surface (Dark)**: `#1e2022`
- **Foreground (Light)**: `#11181C`
- **Foreground (Dark)**: `#ECEDEE`

## デザイン原則
- iOS Human Interface Guidelinesに準拠
- 片手操作を考慮したレイアウト
- 縦向き(9:16)に最適化
- 天気情報が一目で分かるシンプルなデザイン
- ダークモード対応
